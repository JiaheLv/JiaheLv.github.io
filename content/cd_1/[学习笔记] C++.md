`argc`给出传递给程序的参数数量，而`argv`包含这些参数。其中，`argv[0]`可能是程序名称，也可能是空字符串。所以在需要获取程序名称时，不要依赖`argv[0]`，而应该使用特定平台的函数来获取。包含在`argv`中的参数需要从索引`1`开始计算。

### 命名空间

```C++
namespace MyNameSpace {
    //code
}
```

命名空间用来解决不同代码段之间的命名冲突问题。  
要使用命名空间中的成员，可以使用作用域解析运算符`::`。  
同一命名空间内的成员可以直接访问，无需使用作用域解析运算符。  
可以使用`using`声明来引入命名空间中的成员，或者使用`using namespace`来引入整个命名空间。

```C++
using MyNameSpace::MyFunction; // 引入命名空间中的特定成员
using namespace MyNameSpace; // 引入整个命名空间
```

#### 嵌套命名空间

嵌套命名空间是指在一个命名空间内部定义另一个命名空间。每个命名空间用`::`分隔。

```C++
namespace MyNameSpace::Networking::FTP {
    //code
}
```

这种紧凑的语法在C++17以前不可用，必须使用嵌套的命名空间定义：

```C++
namespace MyNameSpace {
    namespace Networking {
        namespace FTP {
            //code
        }
    }
}
```

#### 命名空间的别名

命名命名空间别名可用于为另一个命名空间提供一个新的、可能更短的名称：

```C++
namespace MyNameSpace::Networking::FTP {
    //code
}
namespace FTP = MyNameSpace::Networking::FTP; // 创建命名空间别名
```
