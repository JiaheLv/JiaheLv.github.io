    Carnalitas is a unified sex mod framework for Crusader Kings 3. It provides a set of systems that modders can use for their own projects, such as a database of sex scene events. It also implements features that were common in many CK2 sex mods. All these features can be toggled by players through game rules.  
    > Carnalitas 是一个适用于《十字军之王 3》的统一性爱 MOD 框架。它提供了一系列系统，供 MOD 制作者用于自己的项目，例如性爱场景事件数据库。它还实现了许多《十字军之王 2》性爱 MOD 中常见的功能。所有这些功能都可以通过游戏规则由玩家进行切换。  
    1. Carnalitas

        Carnalitas - Historical Slavery Doctrines, which adds religious doctrines for slavery that are accurate to every faith in vanilla CK3 based on historical research. The submod is separate from the main mod for improved compatibility with total conversion mods.  
        > “Carnalitas - Historical Slavery Doctrines”（肉身 - 历史奴隶制教义 ），它基于历史研究，为原版 CK3 中的所有宗教信仰添加了符合其教义的奴隶制相关教义。该子模组独立于主模组，以提高与大型转换模组的兼容性。
